# PurpleDoc Automation Suite (Modular)
A cleaned, modular, and lint-friendly refactor of the PurpleDoc automation script.

Structure:
```
purpledoc-automation-suite-opt/
├── purpledoc/
│   ├── __init__.py
│   ├── config.py
│   ├── smartsheet_client.py
│   ├── email_client.py
│   ├── parser.py
│   ├── pdf_util.py
│   └── forms.py
├── main.py
├── requirements.txt
└── README.md
```

Run with:
```
python main.py
```
